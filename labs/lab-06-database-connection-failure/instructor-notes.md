# Instructor Notes

## Teaching goal

Teach learners how to verify assumptions about files, folders, database tables and startup scripts.

## Discussion prompts

- What setup steps should happen before an API starts?
- Why are migrations useful?
- What should a helpful database error message include?

## Common learner mistake

Learners may focus on the SELECT query and miss that the database or table doesn't exist yet.

## Production connection

In production apps, database failures may be caused by missing migrations, incorrect credentials, unavailable hosts or bad connection strings.
